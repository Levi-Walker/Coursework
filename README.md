# Computer Science Coursework

This repository contains coursework from my degree at Andrews University.

## Research

- [Brain Tumor Segmentation with U-Net](./Research/) is a replication study evaluating a 2D U-Net on multi-modal brain MRI data, with rotation augmentation and validation-based segmentation threshold tuning.

## Courses

- [Data Science Fundamentals](./Data%20Science%20Fundamentals/) contains introductory work with pandas, data cleaning, web data, visualization, and a Tableau workbook.
- [Data Analytics Development and Integration](./Data%20Analytics%20Development%20and%20Integration/) contains R exercises on data wrangling, joins, web scraping, and plotting.
- [Machine Learning](./Machine%20Learning/) covers preprocessing, classical models, validation, regularization, and introductory neural networks.
- [Mobile App Development](./Mobile%20App%20Development/) contains Android assignments on activities, layouts, fragments, local storage, networking, and a larger pantry project.
- [Networking](./Networking/) contains a UDP chat program and a distance-vector routing simulation.
- [Text and Media Analytics](./Text%20and%20Media%20Analytics/) contains natural language processing and computer vision notebooks.
