# Pantry Tracker

Pantry Tracker is the final version of my course pantry project. It stores pantry items in SQLite and supports adding, editing, deleting, searching, sorting, and filtering items. Items can also have expiration dates, quantities, and camera or gallery photos.

## What I learned

- Organizing SQLite CRUD operations
- Keeping a RecyclerView in sync with stored data
- Passing Parcelable item data between Activities
- Saving sorting and filter choices with SharedPreferences
- Working with dates and expiration indicators
- Handling camera and image picker results
