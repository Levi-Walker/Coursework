# Feature Demos

These are small Android assignments that focus on one or two course topics.

- [Activity Lifecycle](./activity-lifecycle/) logs the main Activity lifecycle callbacks.
- [Animal List](./animal-list/) uses a custom ArrayAdapter with image backed rows.
- [Direction Picker Animation](./direction-picker-animation/) uses alpha animations, delayed callbacks, and interaction state.
- [Fragments and Preferences](./fragments-and-preferences/) switches between Fragments and stores values with SharedPreferences.
- [Job Manager Layout](./job-manager-layout/) practices arranging an interface with ConstraintLayout.
- [Product Registration Database](./product-registration-database/) inserts, queries, categorizes, and deletes SQLite records.
- [Quote of the Day](./quote-of-the-day/) uses a small model class and displays a random quote.
- [SharedPreferences Demo](./shared-preferences/) saves and reads string, integer, and boolean values.
- [SQLite Database Demo](./sqlite-database-demo/) practices SQLiteOpenHelper, inserts, queries, and deletion.
