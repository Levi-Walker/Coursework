# Mobile App Development

This directory contains the Android Studio projects I completed during my mobile app development course. These projects cover core Android application development concepts. Larger assignments are listed at the top level, while smaller exercises and practice apps are organized under [Feature Demos](./feature-demos/).

## Major Projects

* **[Pantry Tracker](./pantry-tracker/)**, My final course project, a pantry management application featuring SQLite storage, search, sorting, filtering, expiration date tracking, and item photos.
* **[Contact Manager](./contact-manager/)**, A contact management app with SQLite persistence, contact editing, dial intents, and photo support.
* **[Animal JSON Browser](./animal-json-browser/)**, An app that retrieves and displays animal records from a JSON endpoint using Volley.
* **[VeggieTales Character Catalog](./veggietales-character-catalog/)**, A character catalog demonstrating custom list views, image resources, dialogs, and a detail activity.

## Skills Demonstrated

* Android application development with Android Studio
* Android Activities, Fragments, and intents
* XML layouts and custom list item views
* Data persistence with SQLite and SharedPreferences
* RecyclerView and ArrayAdapter
* Network requests and JSON parsing with Volley
* Camera and gallery integration for image selection
