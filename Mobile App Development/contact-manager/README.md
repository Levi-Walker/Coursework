# Contact Manager

This Android assignment stores contacts in a local SQLite database and displays them in a RecyclerView. Contacts can be added, edited, deleted, or opened in the phone dialer, and each contact can use a camera or gallery photo.

## What I learned

- Separating SQLite operations into a data access class
- Passing Parcelable objects between Activities
- Handling Activity results
- Building and updating RecyclerView rows
- Using dial, camera, and image picker intents
