# Data Science Fundamentals

This folder contains four guided notebooks and one Tableau exercise from an introductory data science course.

## Assignments

- [Bias–Variance Analysis](./bias-variance-analysis.ipynb) compares linear and polynomial models fitted to mammal weight data.
- [Exploratory Data Analysis and Cleaning](./exploratory-data-analysis-and-cleaning.ipynb) works with missing values, strings, imputation, and plots using flight, building permit, and movie data.
- [Web Table Extraction with pandas](./web-table-extraction-with-pandas.ipynb) reads historical box office tables from HTML and cleans their currency fields.
- [Web Scraping and Text Processing](./web-scraping-and-text-processing.ipynb) parses NPR archive pages with Beautiful Soup and regular expressions.
- [Tableau Music Sales](./Feb22_Lab_Tableau.twbx) is a packaged workbook with a music sales data extract and dashboard.

## What I learned

- Exploring and cleaning data with pandas
- Handling missing and inconsistent values
- Collecting tables and text from web pages
- Comparing underfitting and overfitting
- Creating basic visualizations in Python and Tableau
