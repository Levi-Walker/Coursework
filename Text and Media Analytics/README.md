# Text and Media Analytics

This course covered introductory natural language processing and computer vision. The folder contains seven notebooks and the local media used by the image exercises.

## Natural Language Processing

- [spaCy Natural Language Processing](./spacy-natural-language-processing.ipynb) covers tokens, linguistic attributes, stop words, similarity, dependency parsing, named entities, matching, and sentiment.
- [NLP with Machine Learning](./nlp-with-machine-learning.ipynb) builds a spam classifier with TF-IDF features and a random forest, then saves the model.
- [Help Desk NLP Assistant](./help-desk-nlp-assistant-mini-project.ipynb) combines pretrained sentiment, summarization, and question answering pipelines.

## Computer Vision

- [Digital Image Basics](./digital-image-basics.ipynb) works with image arrays, color channels, resizing, cropping, and rotation.
- [Image Histograms and Equalization](./image-histograms-and-equalization.ipynb) compares grayscale and color histograms and applies histogram equalization.
- [Edge Detection and Segmentation](./edge-detection-and-segmentation.ipynb) uses Sobel, Laplacian, and Canny edge detection with threshold based segmentation.
- [Image and Video Compression](./image-and-video-compression.ipynb) compares JPEG settings, run length encoding, and video compression results.

## What I learned

- Processing and classifying text
- Using pretrained language model pipelines
- Representing and transforming images as arrays
- Detecting edges and segmenting images
- Comparing compression quality and file size

[`cvutils.py`](./cvutils.py) is shared by the image notebooks, and local media is under [`images/`](./images/).
