# Machine Learning

These notebooks are from my machine learning course. They begin with individual algorithms and preprocessing techniques, then move into model comparison, validation, and neural networks.

## Larger exercises

The California housing work is split into three notebooks:

1. [Exploratory Analysis](./california-housing-01-exploratory-analysis.ipynb) examines distributions, geographic patterns, correlations, and stratified sampling.
2. [Data Preparation](./california-housing-02-data-preparation.ipynb) handles missing values, categorical encoding, custom transformations, scaling, and pipelines.
3. [Model Training](./california-housing-03-model-training.ipynb) compares linear regression, decision trees, and random forests with cross-validation and parameter searches.

The [Medical Insurance Cost Modeling](./medical-insurance-cost-modeling.ipynb) notebook applies a similar preprocessing and model comparison workflow to insurance costs.
